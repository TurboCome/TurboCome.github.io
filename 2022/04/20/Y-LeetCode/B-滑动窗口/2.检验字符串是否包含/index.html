检验字符串是否包含__567

2022-04-20 LeetCode►滑动窗口 滑动窗口 0 评论

给定两个字符串 s1 和 s2，写一个函数来判断 s2 是否包含 s1 的排列, 排列可以是任何顺序

换句话说：第一个字符串的 排列之一 是 第二个字符串的子串，任意排列是s2 连续的*子串*

输入: s1 = “ab” s2 = “eidbaooo”

输出: True

解释: s2 包含 s1 的排列之一 ( “ba” )

输入: s1= “ab” s2 = “eidboaoo”

输出: False

注意：

输入的字符串只包含小写字母

两个字符串的长度都在 [1, 10,000] 之间

优化版本：

使用外加 res[26], 前–；后++

依次维护 res ，来和 array 比对

思路：

无序排列的字符串，就是序列字符串，使用 Map | str[ c-‘a’] 来记录

最终判断是否为==0，或 2个Map是否相等
